# Tokenomics de lanzamiento educativo AVICOIN

AVICOIN es un proyecto educativo. No promete precio, rendimiento, apreciación, liquidez ni valor financiero.

| Concepto | Decisión vigente |
|---|---:|
| Nombre / símbolo | AVICOIN / AVI |
| Decimales | 9 |
| Supply inicial autorizado | 1,000 AVI |
| Unidades base iniciales | 1,000,000,000,000 |
| Operaciones `mintTo` de lanzamiento | exactamente 1, sólo con supply 0 |
| Supply máximo permanente | `null` / no definido |
| Mint authority | retenida temporalmente por la wallet de producción |
| Freeze authority | `none` permanentemente |
| Emisiones posteriores | no autorizadas en la etapa actual |

El lote inicial no es un máximo permanente. Conservar la mint authority concentra control y significa que técnicamente podría existir emisión futura, pero cualquier decisión así requeriría una nueva autorización y documentación; la política y las guardas actuales rechazan una segunda emisión. No debe describirse el supply como fijo mientras la autoridad permanezca retenida.

El precio `0.01 USDC/AVI` sólo expresa la proporción inicial del pool educativo propuesto, con máximos de 1,000 AVI y 10 USDC. No es valoración, garantía ni promesa de mercado. Mint, metadata on-chain, supply y pool Mainnet continúan sin crear.
